# TechLog

## 概要
TechLogは、日々の学習内容やエラーの解決過程を記録・管理できるWebアプリケーションです。

学習記録とエラーログを一元管理し、振り返りや問題解決の効率化を目的としています。

## 作成背景
プログラミング学習を進める中で、エラーの内容や解決方法を記録しても、後から見返しづらいと感じていました。

また、単なるメモでは「どのエラーが未解決なのか」「どの学習内容なのか」が分かりにくく、整理された形で管理したいと考えました。

そこで、学習記録とエラーログを分けて管理し、解決状況や公開設定などを持たせることで、より実用的なログ管理アプリを開発しました。

## 主な機能

### 投稿機能
- 学習記録 / エラーログの投稿
- タイトル・本文の入力

### 投稿タイプ管理
- 学習記録とエラーログを分類

### ステータス管理
- 未解決 / 解決済みの管理

### 公開設定
- 投稿の公開 / 非公開の切り替え

### 投稿一覧表示
- 最新順での表示
- 投稿タイプ・ステータスをタグ形式で表示

### 投稿詳細表示
- 詳細情報の表示
- 投稿者のみ削除可能

## 工夫した点
### 状態を視覚的に分かりやすくするUI設計

投稿タイプ（学習記録 / エラーログ）やステータス（未解決 / 解決済み）をタグ形式で表示し、直感的に内容を把握できるようにしました。
また、ステータスごとに色を変えることで、状態が一目で分かるように工夫しました。

### 学習ログとエラーログの分離設計

単なる投稿機能ではなく、「学習記録」と「エラーログ」を分けることで、用途ごとに整理しやすい構造にしました。

### 公開・非公開機能の実装

投稿に公開設定を持たせることで、将来的に他ユーザーと共有できる拡張性を意識した設計にしました。
